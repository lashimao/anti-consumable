# 反耗材白皮书

> 关于人的稀缺性的一份开源文档 · CC BY-SA 4.0

出生，上学，上班，死亡。这条流水线的前提是：社会需要大量可以互相替换的人。

我们管这个叫**耗材循环**。大部分人不是活得失败，是活得可替换。

本仓库是一份退出说明书，和一场公开实验。

## 读什么

- **[MANIFESTO.md](MANIFESTO.md)** —— 宣言（3 分钟）
- **[WHITEPAPER.md](WHITEPAPER.md)** —— 白皮书全文（15 分钟）：三个带时间戳的事实、收窄后的主张、工程化退出协议六条、反方钢人表、公开实验
- **[research/](research/)** —— 全部调研底稿，**含反方弹药**。我们承诺每讲一个赢家案例，同页给失败率
- **[experiments/](experiments/)** —— 实验体登记。作者本人是 #001：21 岁，账面不到一万元，账本公开，2026-09-01 第一次验收，失败数据同样公开

## 三句话立场

1. 认知商品正在率先进入丰裕，稀缺不终结、只转移——向判断、趣味、信任、敢署名处转移。
2. 个体最优策略第一次反转：把可标准化的部分交给机器，把自己压缩到不可替换的那一小块上。
3. 我们用钱验收，不用钱排座次——排行榜取消，仪表盘保留（不可替换度 / 署名率 / 时间主权）。
